# 2D Platformer Game (Godot)

## Overview
This is a basic 2D platformer game developed using the Godot Engine.  
The project was created as part of my learning journey in game development by following a YouTube tutorial.

The goal of this project was to understand core game development concepts such as:
- Player movement and physics
- Scene and node system in Godot
- Collision detection
- Basic game mechanics like coins, enemies, and platforms

## Features
- Player movement (left, right, jump)
- Gravity and collision system
- Tile-based level design
- Collectable coins with scoring system
- Moving platforms
- Basic enemy behaviour
- Kill zone and respawn system
- Camera follow system
- Sound effects and background music

---

## What I Learned
- How Godot’s **node and scene system** works
- Basics of **GDScript**
- Handling **player input and physics**
- Using **signals for interaction (e.g., coin collection)**
- Implementing **animations and UI elements**
- Structuring a small but complete game project

---

## Challenges Faced

- Managing multiple game elements (player, enemies, UI)

- **D / Right Arrow** → Move Right  
- **Space** → Jump
